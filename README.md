# eslint-config

We Make Website's eslint rules and config.

## Installation

Run the command:

```
yarn add --dev eslint @we-make-websites/eslint-config
```

## Usage
We Make Website's eslint rules come bundled in `@we-make-websites/eslint-config`. To enable these rules add the following to your _.eslintrc_ root file:

```js
extends: [
  '@we-make-websites/eslint-config',
],
```
