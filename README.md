# eslint-config

We Make Website's eslint rules and config

## Installation

Install [eslint](https://eslint.org/) and `@we-make-websites/eslint-config`:

```
yarn add --dev eslint @we-make-websites/eslint-config
```

```
{
  "extends": ["@we-make-websites/eslint-config"]
```
